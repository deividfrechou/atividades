O que é Hertz (Hz)?

Hertz não é uma palavra exclusiva de computador. É simplesmente a unidade que mede quantas vezes por segundo alguma coisa se repete. Um coração batendo uma vez por segundo bate a 1 Hz. Um metrônomo marcando duas batidas por segundo está em 2 Hz.

Pensando assim

Dentro do processador, cada "batida" é um pequeno passo de trabalho: somar dois números, mover um dado de um lugar para outro. Um processador de 3 GHz dá 3 bilhões dessas batidas por segundo. É rápido demais pra qualquer coisa que a gente consiga imaginar em escala humana.

UnidadeSignificadoExemplo
1 Hz1 vez por segundoPonteiro de segundos de um relógio
1 MHz1 milhão de vezes por segundoProcessadores dos anos 80/90
1 GHz1 bilhão de vezes por segundoProcessadores atuais (2 a 5 GHz)

E os núcleos (cores)?

Se o Hz é a velocidade de cada "batida", os núcleos são a quantidade de trabalhadores fazendo essas batidas ao mesmo tempo.

Pensando assim

Imagine uma cozinha. Com 1 núcleo, existe apenas 1 cozinheiro: ele corta, tempera e mexe a panela, uma coisa de cada vez — mesmo que seja rapidíssimo. Com 4 núcleos, são 4 cozinheiros trabalhando ao mesmo tempo, cada um numa tarefa diferente. O resultado sai muito mais rápido quando o trabalho pode ser dividido.

E o que são threads?

Pensando assim

Uma thread é um cozinheiro que aprendeu a alternar rapidinho entre duas tarefas, mesmo trabalhando sozinho: mexe a panela um pouco, dá uma olhada no forno, volta pra panela. Ele não virou dois cozinheiros — continua sendo um só — mas aproveita melhor os momentos de espera. Um núcleo com threads consegue lidar com mais de uma tarefa "ao mesmo tempo", mesmo sem ser um núcleo extra de verdade.

Pequenos anéis magnéticos entrelaçados por fios finíssimos; cada anel guardava 1 bit.

Década de 1970

A memória passou a ser fabricada em chips, ficando muito mais rápida, compacta e barata.

Hoje

RAM DDR moderna

Pentes de memória que enviam e recebem dados bilhões de vezes por segundo. Explicamos direitinho em Memória RAM.

Memória RAM: rápida, mas esquecida

A RAM (Random Access Memory, "memória de acesso aleatório") é a memória que o computador usa enquanto está ligado e trabalhando. É nela que ficam os programas abertos, as abas do navegador, o jogo que você está jogando agora mesmo.

Pensando assim

Pense na diferença entre uma mesa de trabalho e um armário. A mesa (RAM) é onde você espalha o que está usando agora — rápido de pegar, rápido de guardar, mas pequena. O armário (armazenamento, que veremos a seguir) guarda tudo por muito mais tempo, só que é mais lento para buscar as coisas lá dentro.

A característica mais importante da RAM é que ela é volátil: quando o computador é desligado, tudo que estava na RAM desaparece. Por isso, quando você "salva" um arquivo, ele sai da RAM (temporária) e vai para o armazenamento (permanente).

Como a RAM funciona por dentro

A RAM é organizada em milhões de "endereços" — como caixinhas de correio numeradas. Cada endereço guarda um pequeno pedaço de dado. Quando o processador precisa de alguma informação, ele simplesmente pede o endereço certo, e a resposta chega quase instantaneamente, porque é tudo eletrônico — não existe nenhuma peça mecânica se movendo, como acontece num HD.

A maior parte da RAM usada hoje (chamada DRAM) guarda cada bit como uma cargazinha elétrica dentro de um capacitor microscópico. O problema é que essa carga vaza sozinha em poucos milissegundos. Para os dados não desaparecerem, o computador precisa recarregar cada capacitor milhares de vezes por segundo, o tempo todo, enquanto está ligado. É justamente essa dependência constante de energia que torna a RAM volátil: assim que a energia falta, esse processo de recarga para, as cargas se esvaem e os dados somem quase instantaneamente — não é só uma questão de "esquecer ao desligar", é literalmente eletricidade vazando.

Como isso aparece no jogo a seguir

Cada par de símbolos que você encontrar no jogo representa um dado guardado num desses "endereços" da RAM, prontinho para o processador usar a qualquer momento — é por isso que, quando você acerta um par rapidamente, parece "instantâneo": é exatamente essa velocidade que a RAM oferece. Só que existe um porém: enquanto você joga, uma barra de energia vai diminuindo, representando essa recarga constante que mantém os capacitores "vivos". Se a energia acabar antes de você organizar tudo, é como desligar o computador de repente: os capacitores param de ser recarregados, a carga vaza e tudo que ainda estava "solto" na memória, sem ter sido combinado (gravado), simplesmente desaparece — é a volatilidade da RAM na prática.

Armazenamento

Bits, bytes e todo o resto

Tudo o que um computador guarda de forma permanente — fotos, músicas, textos — vira, no fundo, uma sequência enorme de 0s e 1s. Cada 0 ou 1 é um bit. Um grupo de 8 bits forma um byte, que costuma dar conta de guardar uma única letra ou caractere.

Analogia · blocos de montar
Pensa em peças de Lego: 1 peça é um bit. Encaixando 8 peças você escreve 1 letra (1 byte). Um punhado de letras forma uma palavra (kilobyte). Muitas palavras formam uma foto (megabyte). Muitas fotos formam um filme (gigabyte).

Como o computador guarda e processa imagens (pixels)

Uma imagem digital é dividida numa grade de quadradinhos chamados pixels. Cada pixel guarda um número (ou um grupo de números) que representa sua cor. Numa imagem simples, em preto e branco, cada pixel pode ser representado por apenas 1 bit: 1 para pintado, 0 para branco. Numa foto colorida, cada pixel costuma usar 24 bits — 3 grupos de 8 bits para dizer quanto de vermelho, verde e azul aquele ponto tem.

Pensando assim

É como preencher uma cartela de ponto-cruz: cada quadradinho da grade é um pixel, e a sequência de instruções ("pinta, pinta, deixa em branco, pinta...") que alguém segue linha por linha é exatamente o tipo de código que o computador guarda para lembrar da imagem depois.

Processar uma imagem é o processador lendo e desenhando esses números na tela, muitas vezes por segundo. Armazenar uma imagem é guardar essa mesma sequência de números permanentemente num arquivo. Quanto mais pixels (maior a resolução) e mais cores, mais bits são necessários — por isso fotos em alta resolução ocupam mais espaço.
